📐 ¿Cuándo se pueden combinar dos términos?

Regla de términos semejantes

✅Dos términos son semejantes si tienen la misma variable con el mismo exponente. Se pueden sumar o restar.
❌Dos términos son no semejantes si tienen variables distintas o exponentes distintos. No se pueden combinar.
✅ Semejantes
3x y 5x
Misma variable x, mismo exponente 1
→ 3x + 5x = 8x ✓
❌ No semejantes
3x y 5y
Variables distintas: x ≠ y
→ 3x + 5y no se simplifica
✅ Semejantes
7a² y −2a²
Misma variable a, mismo exponente 2
→ 7a² − 2a² = 5a² ✓
❌ No semejantes
4x² y 4x
Misma variable pero exponentes distintos: 2 ≠ 1
→ 4x² + 4x no se simplifica
⚠️ Copia también este ejemplo: Gasto semanal de un estudiante de bachillerato:
5t = viajes en metro (t = viajes)
30c = comidas en cooperativa (c = comidas)
5t + 30c → NO se simplifica porque t y c son cosas distintas.
Si intentaras sumarlos obtendrías "35tc" que no tiene sentido real.

Regla clave para el examen

¿Cuándo SÍ se combinan? Cuando tienen la misma variable con el mismo exponente.
Ej: 3x² + 7x² = 10x²
¿Cuándo NO se combinan? Variable distinta o exponente distinto.
Ej: 3x² + 7x → se quedan como 3x² + 7x (binomio)
📌 Copia también: El número de términos de una expresión determina su nombre, pero solo después de combinar todos los términos semejantes. Primero simplifica, luego clasifica.
